# Step 07 — Drag & Drop (Lists + Cards)

## Goal
Enable drag‑and‑drop for lists and cards.

## Tasks
1. Install react-beautiful-dnd or @dnd-kit.

2. Wrap board in DragDropContext.

3. Implement list drag:
   - Horizontal movement
   - reorderLists()

4. Implement card drag:
   - Vertical movement
   - Cross-list movement
   - moveCard()

## Output
Smooth drag‑and‑drop for lists and cards.
